# Thai warm-moment audio

Drop recorded clips here, named exactly as `phrases.js` expects:

- `dodge_th.mp3`
- `praise_th.mp3`
- `caught_th.mp3`
- `tomorrow_th.mp3`

`app.js` tries to play each one and falls back to English TTS automatically
if the file is missing or fails to load — so the game works fine before any
of these are recorded.
